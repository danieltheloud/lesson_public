# 사용자 및 권한

## 사용자(user)

- 사용자는 DBMS 서버에 접속해서 각종 쿼리를 실행할 수 있는 주체를 의미한다.
- DBMS 설치 시, 최고 관리자 권한을 가지는 사용자인 `root`가 존재하게 된다.
