# 규칙

- 쿼리(`query`)는 최소한의 실행 단위이고, 세미콜론(`;`)으로 구분한다.
- 쿼리 작성시에 키워드의 경우 전체 **대문자**(혹은 소문자)로 통일하여 작성한다.
- 문자열 리터럴의 경우 홑따옴표(`'`)로 감싼다.
- 구성요소의 이름은 백틱(`` ` ``)으로 감싼다.

## 주석

- 주석(`comment`)은 쿼리 실행에 영향을 미치지 않고 간단한 정보를 남기기 위해 사용한다.
- 한 줄 주석은 샾(`#`) 혹은 하이픈 두 개(`--`)로 작성한다.
- 여러줄 주석은 `/*`로 시작하여 `*/`로 끝낸다.
