# list: 리스트

- 리스트 자료형은 쉼표(`,`)로 구분된 항목을 대괄호(`[`, `]`)로 감싸서 표현한다.
- 빈 리스트는 한쌍의 대괄호로 표현된다.
- `Python`의 리스트 자료형은 각 요소의 자료형이 서로 다를 수 있다.

## 요소 삭제

- `del`
  - `del x`
- `append()`
  - `append()`
- `insert()`
  - `insert(x, y)`
- `remove()`
  - `remove(x)`
- `pop()`
  - `pop()`
  - `pop(x)`
- `extend()`
  - `extend(x)`
