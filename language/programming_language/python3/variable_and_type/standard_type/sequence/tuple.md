# `tuple`: 튜플

- 쉼표(`,`)로 구분된 항목을 소괄호(`(`, `)`)로 감싸서 표현한다.
- 튜플은 수정이나 삭제가 불가능하다는 점 외에 리스트와 유사하다.
- 빈 튜플은 한쌍의 소괄호로 표현된다.
- 요소가 하나뿐인 튜플의 리터럴은 요소의 뒤에 쉼표를 반드시 붙여줘야 한다.
- 쉼표로 구분된 여러 요소들을 하나의 변수에 할당하면, 튜플로 간주된다.

  ```py
  myTuple = (1, 3, 5, 7)
  emptyTuple = ()
  singleElementTuple = (10,)
  anotherTuple = 2, 4, 6, 8
  ```
