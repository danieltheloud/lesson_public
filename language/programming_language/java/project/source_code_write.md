# 소스 코드 작성

- 프로젝트를 생성하면 `src`라는 이름의 폴더가 자동으로 만들어진다.
  - `src` 폴더 하위에 `App.java`라는 파일이 존재하며, 아래와 같은 코드가 나타난다.

    ```java
    public class App {
        public static void main(String[] args) throws Exception {
            System.out.println("Hello, World!");
        }
    }
    ```

  - `F5`키를 누르면, `main` 함수에 작성된 코드가 실행된다.
